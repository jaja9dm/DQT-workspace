# DQT — Daily Quant Trading System

국내 주식 단타 자동매매 시스템. Claude AI 기반 멀티팀 아키텍처.

---

## 개요

KIS API + FinanceDataReader + Claude (Sonnet/Opus/Haiku)를 조합하여
장 중 자동으로 종목을 스캔하고, Hot List를 선정하고, 매수·감시·청산까지 실행하는 퀀트 시스템.

```
실시간 시장 데이터 → 유니버스 스캔(450종목) → Claude Hot List 판단
    → 5단계 게이트 → 분할 매수 → 트레일링 스톱 + 분할 익절
    → 텔레그램 알림 → 일일 리포트
```

## 아키텍처

| 팀 | 역할 | 주기 |
|----|------|------|
| 글로벌 시황팀 | S&P500·VIX·환율 분석, 글로벌 리스크 점수(0~10) | 1시간 |
| 국내 시황팀 | KOSPI/KOSDAQ 지수·투자자동향·뉴스 분석, 시황 점수(-1~+1) | 30분 |

| 장중 MACD팀 | Hot List·보유 포지션 분봉 MACD Pre-Cross 감지 | 3분 |
| 위기 관리팀 | 포트폴리오 손익·시황 종합 → 리스크 레벨(1~5) | 15분 |
| 포지션 감시팀 | 보유 잔고 실시간 감시, 손절·익절·타임컷 자동 실행 | 90초 |
| 매매팀 | 5단계 게이트 → Claude 최종 판단 → KIS 주문 실행 | 5분 |
| 리포트팀 | 일일 성과 집계·텔레그램 발송 | 15:40 배치 |
| 연구소 | 전략 성과 분석·파라미터 자동 조정·백테스트 | 16:00/일요일 |

## 매매 전략

### 진입 조건 (5단계 게이트)
1. **리스크 레벨** ≤ 3 (Level 4↑ 진입 차단)
2. **글로벌 시황** outlook != 'negative'
3. **국내 시황** 점수 ≥ -0.3
4. **Hot List** 포함 종목 (최근 10분 이내)
5. **Claude 최종 판단** (감성 분석 + 시황 + 신호 종합)

### Hot List 선정 기준
- 거래량 급증 (평균 3배↑) + 가격 상승 동반
- 볼린저밴드 상단 돌파 + RSI 60~70
- MACD 히스토그램 수렴 (골든크로스 임박)
- 일봉 MACD 필터: 비강세 종목 원천 제외

### 포지션 관리
- **분할 매수**: 1차 40% → 2차 35% → 3차 25% (-1% 추가 하락 시)
- **1차 익절**: +5% → 1/3 매도
- **2차 익절**: +10% → 1/3 매도
- **트레일링 스톱**: 수익 +10% 도달 시 손절선 상향 (현재가의 -5%)
- **초기 손절선**: -5% (기본값, `.env`에서 조정)
- **타임컷**: 5 영업일 초과 전량 청산
- **오프닝 게이트**: 9:00 직후 시황 평가 → 즉시 매수 또는 9:10 대기

## 설치 및 실행

```bash
# 의존성 설치
pip install -r requirements.txt

# 환경 변수 설정
cp .env.example .env
# .env 편집: KIS API 키, Anthropic API 키, 텔레그램 봇 토큰 입력

# 실행 (스케줄러 기동)
python main.py

# 즉시 실행 (개발·테스트용 — 장 시간 무관)
python main.py --now
```

## 환경 변수

| 변수 | 설명 | 기본값 |
|------|------|--------|
| `KIS_MODE` | `paper` (모의) / `live` (실거래) | `paper` |
| `KIS_APP_KEY` | KIS Open API 앱키 | 필수 |
| `KIS_APP_SECRET` | KIS Open API 시크릿 | 필수 |
| `KIS_ACCOUNT_NO` | KIS 계좌번호 (`XXXXXXXX-XX`) | 필수 |
| `ANTHROPIC_API_KEY` | Anthropic API 키 | 필수 |
| `TELEGRAM_BOT_TOKEN` | 텔레그램 봇 토큰 | 필수 |
| `TELEGRAM_CHAT_ID` | 텔레그램 채팅 ID | 필수 |
| `TRAILING_INITIAL_STOP_PCT` | 초기 손절선 (%) | `5.0` |
| `TRAILING_TRIGGER_PCT` | 트레일링 시작 수익률 (%) | `10.0` |
| `TRAILING_FLOOR_PCT` | 트레일링 간격 (%) | `5.0` |
| `TAKE_PROFIT_1_PCT` | 1차 익절 수익률 (%) | `5.0` |
| `TAKE_PROFIT_2_PCT` | 2차 익절 수익률 (%) | `10.0` |
| `MACD_DAILY_FILTER` | 일봉 MACD 필터 활성화 | `true` |
| `MACD_EARLY_EXIT_ENABLED` | MACD 조기손절 활성화 | `true` |

## 기술 스택

- **언어**: Python 3.9+
- **AI**: Claude Sonnet 4.6 (매매·시황), Claude Haiku 4.5 (감성캐시), Claude Opus 4.6 (연구소)
- **데이터**: KIS Open API, FinanceDataReader, yfinance, FRED API
- **DB**: SQLite (WAL 모드, 13개 테이블)
- **스케줄러**: APScheduler
- **알림**: Telegram Bot API
